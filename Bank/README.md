# Bank Management System (C++)

A console-based bank management system written in C++.
This project demonstrates incremental feature development, and basic system design.

## Project Structure

- **v1-basic/**
  - Client management
  - Deposit and withdraw transactions

- **v2-advanced/**
  - User login system
  - Admin and user roles
  - Permission-based access control
  - User management (add/delete users)

## Features

- Add, update, and delete bank clients
- Deposit and withdraw money
- Transaction handling
- User authentication
- Role-based permissions
- Admin management

## Technologies Used

- C++
- File handling
- Console application

## Project Evolution

This project evolved through multiple iterations.
The earlier version focused on core banking operations,
while the advanced version added authentication,
user roles, and permission management.

## How to Run

1. Open the project in any C++ compiler (Visual Studio / Code::Blocks)
2. Build and run the project
3. Follow the console menu instructions
