## Description

The ATM system represents the customer-facing
side of a banking system. It is designed to work
independently from the admin system and focuses
on simplicity and clarity of logic.

This project demonstrates:
- Control flow and menu-based systems
- File handling
- Separation of responsibilities
- Real-world banking workflow simulation
